# Ruby script for converting degrees Celsius to Fahrenheit and Kelvin.

## Celsius
  Input: 36c
*  °F: 52
*  °K: 309

## Fahrenheit
  Input: 32f
*  °C: 0
*  °K: 273

## Kelvin
  Input: 273k
*  °C: 0
*  °F: 32

## Errors
  Input: 36d
  Wrong temperature sign: `d`

  Input: a
  Wrong temperature sign: `a`

  Input: 3
  Wrong input: `3`
